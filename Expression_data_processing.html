As the same gene can correspond to several different probes and therefore, to have several expression data, we will locate those probes that correspond to the same gene, will make the average to the expression data, and will assign this average per patient to the gene. Finally we will obtain two data frames: in one of them duplicated genes with the mean of their data will be put, and another one with non duplicated genes data. We will join the two data frames in one to finish.

Obtaining the data frame with the duplicated genes mean:

library(dplyr)
##

##     intersect, setdiff, setequal, union
duplicates <- data_sondgen_final[duplicated(data_sondgen_final$SYMBOL),1]  #The function "duplicated" takes those duplicated genes and stores only the copies names, that is, if Y is duplicated 3 times (Y, Y, Y), this code will only save the last two repetitions since the first one is the original and does not consider it repeated.
duplicates_unic <- duplicates[!duplicated(duplicates)]  #This time we use negated form of function "duplicated" for last vector created to obtain the repeated genes names only once (and not all the repetitions as copies of them).
media.gen.paciente <- data.frame()  #This will be the dataframe with those duplicated genes with the means of their copies.

for (gen in (seq(duplicates_unic))){  #Looping through each duplicated gene.
  fila <- match(duplicates_unic[gen],data_sondgen_final$SYMBOL) #We get the row where this duplicated gene is found in data_sondgen_final.
  mean_gen <- data_sondgen_final %>% filter(SYMBOL == data_sondgen_final[fila,1])  #mean_gen will get that gene plus all its copies with all its expression data.
  mean <- data.frame(t(colMeans(mean_gen[2:101])))  #mean will save the mean per column (per patient) of the copies for that gene, but without the gene name.
  gen.sindup <- data.frame(mean_gen[1,1],mean)  #gen.sindup is the data frame that stores both the gene name and the average obtained in last step.
  gen.sindup <- rename(gen.sindup,SYMBOL=mean_gen.1..1.)  #In previous gen.sindup, genes' variable name varies from the original (which is SYMBOL), so it must be renamed to be able to work with this data frame later.
  media.gen.paciente <- rbind(media.gen.paciente,gen.sindup)  #gen.sindup is introduced to the previously initialized data frame, that is, the gene with patients' expression data mean.
}

Obtaining the data frame of the non duplicated genes and their expression data:

genesinduplic <- data.frame() #This data frame will have those genes (and their patients) that were not in duplicates_unic.

for (gen in (seq(nrow(data_sondgen_final)))){ #We loop through each gene in data_sondgen_final.
